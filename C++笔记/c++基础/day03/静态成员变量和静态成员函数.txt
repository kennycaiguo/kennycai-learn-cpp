类的静态变量不属于类创建的对象。类的静态变量一定要在类的外部初始化。但是类的对象是可以访问类的
静态变量，类的静态变量可以使用类名::静态变量的发生访问。也可以说类的静态变量是所有的类对象共享的。

如：
class Myclass
{
   public：
      Myclass{}
      Myclass(int a)
   {
       m_a=a;
    }
   static int m_c;
   private：
       int m_a;
};

int Myclass::m_c=10; 

注意，如果将一个静态变量设置为私有，就必须要有一个public 的静态方法来获取这个静态变量的值。
同理，类的静态成员函数不属于类创建的对象。类的静态成员函数是所有的类对象可以使用的。
